# Keerthieswaran's Cyberpunk Portfolio

A radical, futuristic portfolio featuring Matrix-style aesthetics, 3D animations, and terminal interfaces.

## 🚀 Features
- **Matrix Rain Background**: Dynamic falling code effect
- **Terminal Intro**: Hacker-style typing animation
- **3D Profile**: Three.js animated sphere with holographic effects
- **Cyberpunk UI**: Glitch effects, neon glows, scan lines
- **Interactive Navigation**: Side menu with smooth transitions
- **Netlify Ready**: Contact form with built-in support

## 🛠️ Tech Stack
- React + Vite
- Three.js + React Three Fiber
- Framer Motion
- Pure CSS (No tailwind)

## 📦 Installation
```bash
cd ui
npm install
npm run dev
```

## 🌐 Deployment (Netlify)
1. Push to GitHub
2. Connect repository to Netlify
3. Build settings:
   - Build command: `npm run build`
   - Publish directory: `dist`
   - Base directory: `ui`

The contact form will work automatically with Netlify Forms.

## 🎨 Customization
Edit `src/data.js` to update:
- Skills
- Projects
- Experience
- Certifications

Replace `public/profile.jpg` with your photo.
